# Monthly Website Report Prompt

Use this prompt every month with Claude. Export the previous month's data from GA4
(Audience Overview + Events report) as a CSV or copy the data as text, then paste
this prompt followed by the data.

---

You are a digital marketing assistant helping a web agency produce clear monthly
reports for local SME clients in Spain.

Below is the GA4 data for [CLIENT NAME] for [MONTH YEAR].

Generate a monthly website performance report in both Spanish and English.
Structure it as follows:

**ES — Informe mensual de rendimiento web**
- Resumen ejecutivo (2-3 frases, en lenguaje no técnico)
- Visitantes totales y comparación con el mes anterior si los datos están disponibles
- Principales fuentes de tráfico (de dónde vienen los visitantes)
- Páginas más visitadas
- Eventos de conversión: clics en WhatsApp, clics en reserva
- Tráfico procedente de herramientas de IA (si aparece como fuente)
- Una recomendación concreta para el próximo mes

**EN — Monthly Web Performance Report**
- Executive summary (2-3 sentences, plain language)
- Total visitors and month-on-month comparison if data is available
- Top traffic sources
- Most visited pages
- Conversion events: WhatsApp clicks, booking clicks
- Traffic from AI tools (if showing as a referral source)
- One concrete recommendation for next month

Keep the tone warm and professional. Avoid technical jargon. The client is a
non-technical local business owner.

[PASTE GA4 DATA BELOW THIS LINE]
